# 协作目标画像模板

本文件是公开 Skill 的通用模板，不应写入具体用户身份、私人研究内容或未公开目标。

## 读取顺序

1. 优先读取工作区本地 `data/profile/collaboration-profile.md`。
2. 如果本地画像不存在，只使用当前对话中用户明确表达的目标。
3. 不从文件名、历史仓库或其他材料推测用户身份与偏好。

## 建议记录字段

- 研究目标：希望长期解决的问题。
- 内容范围：优先与排除的技术领域。
- 来源标准：官方、论文、社区或其他证据的使用边界。
- 输出形式：长度、语言、结构和文件格式。
- 质量门槛：评分、验证和降级规则。
- 本地/公开边界：哪些产物仅保留本地，哪些允许发布。
- 当前系统状态：已经具备的能力与下一阶段重点。
- 变更日期与原因：仅在用户明确改变目标时更新。

## 定向建议规则

定向建议只能引用本地画像或当前对话中已经确认的目标。每条建议写清当前情境、建议动作、预期收益、成本与风险、优先级、置信度、验收标准和建议结论，并保留用户最终决定权。
