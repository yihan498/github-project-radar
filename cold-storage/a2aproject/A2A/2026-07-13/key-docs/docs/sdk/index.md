# A2A SDK

The following table lists the supported languages for official A2A SDKs and their repositories.

| Language   | Repository                                               |
|------------|----------------------------------------------------------|
| Python     | [`a2a-python`](https://github.com/a2aproject/a2a-python) |
| Go         | [`a2a-go`](https://github.com/a2aproject/a2a-go)         |
| Java       | [`a2a-java`](https://github.com/a2aproject/a2a-java)     |
| JavaScript | [`a2a-js`](https://github.com/a2aproject/a2a-js)         |
| C#/.NET    | [`a2a-dotnet`](https://github.com/a2aproject/a2a-dotnet) |
| Rust       | [`a2a-rs`](https://github.com/a2aproject/a2a-rs)         |

The A2A project provides numerous samples across supported languages in the [a2a-samples repository](https://github.com/a2aproject/a2a-samples).
